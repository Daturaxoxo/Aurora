## What is Smart App Control?
> [!NOTE]
> [Smart App Control](https://learn.microsoft.com/en-us/windows/apps/develop/smart-app-control/overview) is a system introduced by Microsoft to prevent unknown applications from running on your computer. Although the system sounds good on paper, the way Microsoft implemented it will cause issues for Aurora.
> Microsoft either requires one of two things for an app to be ran if Smart App Control is enabled on the device:
> - Either have their cloud-powered security service do an analysis on the file before running, which won't work if you're offline or the file is too new
> - Or, the app must have a Code Certificate.
>
> Code Certificates cost money. A lot of it actually. A premium one can go for $100 per year, basic ones go for about half. Aurora doesn't have the budget and resources for this.
